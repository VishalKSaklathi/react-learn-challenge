Steps to create a new React Project:
-npm create vite@latest
-cd newproject
-npm i
-npm run dev

to add tailwind css:
-npm install -D tailwindcss postcss autoprefixer
-npx tailwindcss init -p
- add this to tailwind.config.js
content: [
    "./index.html",
    "./src/**/*.{js,ts,jsx,tsx}",
  ],
-and this one to index.css
@tailwind base;
@tailwind components;
@tailwind utilities;
